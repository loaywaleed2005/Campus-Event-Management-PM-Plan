# Campus Event Management System – Project Management Plan

A complete Agile project management plan for a web-based campus event management system developed at the German International University.

## Project Overview

The proposed platform provides a centralized system where:

- Students can discover events and RSVP
- Organizers can create and manage events
- Administrators can monitor engagement and generate reports

## Project Management Approach

The project follows Agile Scrum using three delivery sprints.
